Description

This application allows you to survey land-surfaces with your android device (android >8.0). This data will be converted into a digital raster model which is useful for land analysis/simulation/design. Those raster models can be exported and for example, used in QGIS

supported inputs:

  • internal GPS
  • Bluetooth GNSS (RTK) receiver (NMEA)
  • USB-Serial GNSS (RTK) receiver (NMEA)
  • GPX file
  • CSV files (EPSG:4326) with latitude, longitude, and elevation

supported outputs:

  • GPX file
  • Ascii Grid (EPSG:3857, EPSG:4326, UTM)
  • image (.png) and world file (.pgw)
